# example-app

This template now showcases the `vue3-excel-table` component with two routes:

- `/` &mdash; A basic table backed by static in-memory data.
- `/rest-api` &mdash; A simulated REST-backed table that talks to a fake API (`src/services/fakeEmployeeApi.ts`). Every create, update, or delete request waits 500â€“900&nbsp;ms so you can observe asynchronous sync states, new employees are added by editing the empty row at the bottom (edits are queued until the API returns an ID), and API calls only fire once you commit a cell (press Enter/Tab or blur).

## Type Support for `.vue` Imports in TS

TypeScript cannot handle type information for `.vue` imports by default, so we replace the `tsc` CLI with `vue-tsc` for type checking. In editors, we need [Volar](https://marketplace.visualstudio.com/items?itemName=Vue.volar) to make the TypeScript language service aware of `.vue` types.
